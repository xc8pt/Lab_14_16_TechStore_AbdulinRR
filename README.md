## **Описание проекта**
Многостраничный сайт интернет-магазина электроники "TechStore" с адаптивной
вёрсткой.

---
Адаптивный лендинг интернет-магазина электроники **TechStore** с 3 страницами:
- **Главная** (`index.html`) - Hero + популярные товары
- **Каталог** (`catalog.html`) - 9 карточек товаров 3x3
- **О нас** (`about.html`) - преимущества + команда

## **Функционал**
- [x] **Полная адаптивность** (десктоп/планшет/мобильный)
- [x] **Навигация** между всеми страницами
- [x] **Hover-эффекты** карточек и кнопок
- [x] **Анимации** появления элементов
- [x] **Рейтинг товаров** ★★★★★ с голубыми плашками
- [x] **Современный дизайн** с градиентами

## **Реализованные страницы**
- **Главная** - приветственный баннер, популярные товары, преимущества
- **Каталог** - сетка из 9 карточек товаров с фильтрами
- **О нас** - информация о магазине и команде

## Использованные **технологии:**
- **HTML5**
- **CSS3** (`Flexbox`, `Grid`, `Media`, `Queries`)
- **Git/GitHub**
